Práctica 2:
Visión por Computador

Python + FastApi + OpenCV + ESP32-CAM

Autores: Jorge Cueva   |   Kevin Yansaguano

Descripción del Proyecto

▼

Este proyecto integra un servidor desarrollado con Flask para incorporar las imagenes recibidas desde una camara de una placa ESP32-CAM, y procesarlas utilizando OpenCV. La aplicación permite la transmisión en tiempo real de video capturado por la ESP32-CAM, y realiza procesamiento de visión por computador desde las categorías de eliminación de fondo, muestra de fps, además de filtros y operaciones, generación de ruido, gaussiano y speckle, filtros, reducción de ruido, y 2 algoritmos de segmentación de bordes.

Objetivos de la Práctica

▼
  • Generar una página web que permita una interfaz a los filtros y uso de la imagen recibida
  • Realizar la conexión y transmisión de video desde la ESP32-CAM hacia el servidor
  • Procesar los fotogramas capturados utilizando OpenCV
  • Aplicar técnicas de visión por computador desde el servidor python
  • Integrar correctamente Flask como intermediario entre el hardware y el procesamiento

Herramientas Utilizadas

▼
  • Python 3
  • Flask
  • OpenCV
  • ESP32-CAM
